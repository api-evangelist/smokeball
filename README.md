# Smokeball (smokeball)

Smokeball is legal practice management software with a REST API for managing matters, contacts, documents, time entries, billing, trust accounting, and law firm workflows. The API follows REST principles, accepts JSON, returns JSON, and uses OAuth 2.0 authentication. Available across US, AU, and UK regions.
